Case Study: Guest House Webpage

Building a web application providing complete information about the offered services in a fast and functional way.

Guest House Meraklii

  • Industry: Tourism
  • Size: 10+
  • Website: www.meraklii-smilyan.com
Guest House located in Smilyan, Smolyan, Bulgaria.

Project Requirements

  • Quick and functional access to the sections, with responsive design.
  • Complete business information included.
  • A rich gallery giving complete picture of the house.
  • A functional contact section making business easily discoverable and accessible.
  • Good google search performance.

Project Overview

The guest house is located in the village of Smilyan, Smolyan region. The business has well-maintained social networks that present some of the offered services, but they do not have their own web page that provides complete information about the business. The house has several separate parts that have been taken into account, such as bedrooms, tavern, barbecue, relaxation area and children's corner. The web application provides complete information about the services offered, in a fast and functional way, giving a complete picture of the house.

The Challenge

The client's requirement for a rich gallery with quality photos certainly influenced the overall performance of the app. Optimizing the size of the images without compromising their quality was an important part of the process. Building a static website without using a backend and database would be the easiest way to reach the project goals, but it would be a problem to build a functional contact form. A good Google ranking is also an important part of the app that should have been paid attention to.

The Approach & Solution

To gain greater authenticity, a house logo has been created, maximally matching the web application and the client's requirements, through the web application https://www.brandcrowd.com. For better app performance with the large amount of quality images, each one was processed and compressed using https://compressor.io. However, the loading of the gallery tab was not satisfying, therefore resized copies of all the photos were made, used for previews of the various galleries. Sending emails through the contact form using only frontend technologies was not possible, an external web application service https://www.emailjs.com was used to achieve this goal. A sitemap file has been created with https://www.mysitemapgenerator.com for better Google search performance. The web app is hosted via https://bcloud.club services.

Web application is a static website developed entirely on frontend technologies as JavaScript, HTML and CSS. External applications were used to achieve some of the client's requirements.

The Results

The web app performance results were obtained after automated tests through PageSpeed Insights web app.

Performance
89%
Encode images efficiently, Size images correctly, Defer loading images outside the visible part of the screen.
Accessibility
91%
The document contain all needed elements, all elements have valid and properly used attributes.
Best practices
96%
Valid SiteMap, HTTPs used, Avoids third-party cookies, HTML doctype used and etc.
SEO
93%
Meta description, Indexing not blocked, link text is descriptive, valid HTTP status.

The presence of a large amount of high-quality photos in gallery reduces the performance rate.
